# Portfolio Build Phases

## Phase 0 — Foundation & Style Guide
Goal: Create the project foundation before building pages.

Includes:
- Vite/React setup
- Git/GitHub setup
- CSS architecture
- design tokens
- style guide page
- design-system documentation
- CSS philosophy documentation

Status: complete / mostly complete

---

## Phase 1 — Hero & Site Identity
Goal: Establish the first real visual direction.

Includes:
- homepage hero
- header/nav
- CTA hierarchy
- typography direction
- background image treatment
- color identity
- initial responsive behavior

Status: in progress / nearly complete

---

## Phase 2 — Portfolio Content Strategy
Goal: Decide what the site is actually saying.

Includes:
- final positioning
- homepage narrative
- project categories
- work/system case study themes
- what to anonymize
- résumé alignment
- voice/tone guidelines

---

## Phase 3 — Selected Work & Systems
Goal: Build the proof section.

Includes:
- work cards
- internal systems cards
- project summaries
- tags
- “View case study” placeholders
- screenshots/placeholders

Likely cards:
- Union Website & CMS Builds
- QA Workflow Overhaul
- Design System & Component Library
- SOP & Workflow Modernization

---

## Phase 4 — Skills, Process & Differentiators
Goal: Show how Heather works.

Includes:
- skills/focus section
- process section
- CMS/system thinking
- accessibility/QA emphasis
- workflow improvement
- collaboration and documentation

---

## Phase 5 — About & Contact
Goal: Add human context without overexplaining.

Includes:
- about section
- contact CTA
- résumé link
- LinkedIn/GitHub/email placeholders
- optional values/principles section

---

## Phase 6 — Polish & QA
Goal: Make the site feel finished.

Includes:
- responsive pass
- accessibility pass
- link checks
- spacing polish
- browser checks
- image optimization
- content cleanup
- remove unused CSS

---

## Phase 7 — Deployment
Goal: Publish when ready.

Includes:
- GitHub repo cleanup
- README
- Netlify or Vercel setup
- custom domain later if desired
- final production build check
